I've written many open-source libraries and gems, and have poked around with other projects, but these are perhaps the highlights.

Flying Sphinx

A Heroku add-on for the Sphinx full-text search tool, which I've been running since 2011.

Thinking Sphinx

An open-source library for using the Sphinx full-text search tool with Ruby on Rails applications (and ActiveRecord in particular). I began this project in 2007, and it's still under active development.

Sliver

An open-source library for building clean, simple and object-oriented Rack APIs with Ruby.

Combustion

An open-source library for testing Ruby on Rails Engines somewhat elegantly.

Gutentag

An open-source library for tagging ActiveRecord objects in Ruby on Rails applications, with a focus on simplicity. The code for this project was initially written as an example for a RailsConf workshop on writing gems that I ran in 2013.
